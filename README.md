# plant-taxonomy-field-guide

Offline-first Expo Router MVP for identifying and browsing Illinois plants.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npx expo start
```

## Structure

```text
app/
src/
  data/
  types/
  utils/
  components/
  screens/
```

This scaffold intentionally excludes any backend or authentication setup.
